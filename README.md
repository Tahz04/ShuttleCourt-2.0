# ShuttleCourt - Hệ Thống Đặt Sân Cầu Lông Trực Tuyến

## 📋 Mô Tả Dự Án

**ShuttleCourt** là hệ thống phần mềm hỗ trợ tìm kiếm và đặt sân cầu lông trực tuyến. Nền tảng cho phép người chơi dễ dàng tìm sân, xem lịch trống theo từng khung giờ và thực hiện đặt sân một cách nhanh chóng.

Bên cạnh đó, hệ thống cũng cung cấp các công cụ hỗ trợ quản lý sân, lịch đặt và thông tin khách hàng dành cho chủ sân hoặc quản trị viên. Việc số hóa quy trình đặt sân giúp giảm thiểu tình trạng trùng lịch, nâng cao hiệu quả quản lý và cải thiện trải nghiệm của người dùng.

Hệ thống được thiết kế theo kiến trúc client–server, trong đó ứng dụng di động đóng vai trò giao diện tương tác với người dùng, còn backend cung cấp các dịch vụ API để xử lý dữ liệu và quản lý nghiệp vụ.

## 🎯 Mục Tiêu Hệ Thống

- Xây dựng nền tảng đặt sân cầu lông trực tuyến dễ sử dụng  
- Hiển thị tình trạng sân và khung giờ trống theo thời gian thực  
- Hỗ trợ người chơi tìm sân và đặt sân nhanh chóng  
- Giúp chủ sân quản lý lịch đặt và hoạt động kinh doanh hiệu quả  
- Cung cấp hệ thống đánh giá và phản hồi chất lượng sân

- 🛒 Đặt sân trực tuyến theo khung giờ mong muốn

- 📊 Quản lý sân, lịch đặt và đơn hàng (Admin/Chủ sân)  
- 📱 Giao diện tối ưu cho thiết bị di động

## 🛠️ Công Nghệ Sử Dụng

- **Frontend (Mobile App):** Flutter  
- **Backend (API):** Nodejs
- **Database:** MySQL / PostgreSQL  
- **Authentication:** Laravel Sanctum / JWT  
- **API Architecture:** RESTful API  
- **Deployment:** VPS / Docker  
- **Version Control:** Git
